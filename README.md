# نظام حجز القاعات (Halls Booking System API)

نظام متكامل لإدارة حجز القاعات مبني باستخدام Laravel 11 و Sanctum للتوثيق.

## المميزات (Features)

- نظام مستخدمين متطور: دعم أدوار المالك (Owner) والعميل (Customer).
- توثيق آمن: استخدام Laravel Sanctum لإصدار التوكنز وحماية الروابط.
- إدارة القاعات: نظام لإدارة الأسعار، الصور الأساسية، وتوافر القاعة.
- بيانات تجريبية: نظام Seeding كامل لملء القاعات والمستخدمين فوراً.

## طريقة التشغيل (Setup)

1. قم بتنصيب الحزم: composer install
2. إعداد قاعدة البيانات في ملف .env.
3. تشغيل الـ Migrations والـ Seeds:
   `bash
   php artisan migrate:fresh --seed
